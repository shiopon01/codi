# diagram creater in console

golangの学習の一環として、shiopon01が作成中。  
Markdownでこういう微妙な図作りたいけどめんどくさい時にどうぞ。

## how to use

`+->` `<->` `<-+`の3つのトークンで、文字列を繋ぐ。  
`|` を挟むことで、文字の改行が可能。

```
$ codi "Router +-> Gateway <-> Internet|or|nCorporate network"
+----------+     +-----------+     +---------------------------------+
|  Router  | +-> |  Gateway  | <-> |  Internet|or|Corporate network  |
+----------+     +-----------+     +---------------------------------+
```
